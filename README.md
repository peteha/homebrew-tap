# Peteha Tap

## How do I install these formulae?

`brew install peteha/tap/<formula>`

Or `brew tap peteha/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "peteha/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
